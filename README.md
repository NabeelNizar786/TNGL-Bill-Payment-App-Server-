# TNGL Payment App

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/NabeelNizar786/TNGL-Bill-Payment-App-Server-.git

2. Install npm modules for both client and server:
cd client
npm install

cd ..
npm install

3. Usage
Operator Side (Admin)

1. Run the server:
cd ..
npm start

http://localhost:3000/admin

2. Perform admin tasks such as adding, deleting, and updating user details.

User Side (Customer)

1. Run the client application:

cd client
npm start

4. Open the User Dashboard in your browser:
http://localhost:3000/login

5.Log in to view and edit your profile details.

Navigate to the Bills section to view and pay pending bills.

Important Note
Ensure both the server and client are running concurrently for full functionality.
